## Image Specifications

- **Format**: JPG or WebP (WebP recommended for smaller file sizes)
- **Hero Image**: 1920x1080px or larger
- **Day/Gallery Images**: 1200x900px (4:3 aspect ratio)
- **File Size**: Optimize to <500KB each for faster loading

## Quick Start

1. Add your images to this folder with the exact filenames listed above
2. The HTML file will automatically reference them
3. Deploy to Vercel - images will be served automatically
